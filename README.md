# Chronos
Chronos is fullstack event calendar application. It provides usefull tools for managing own calender, sharing it with others and keep track of tasks, arrangements or just creating yourself reminders with ease.

## Features
- **User Authentication** - Registration, login, email confiramtion, password reset
- **Calendar and Event management** - Create, edit, delete and share your calendars
- **Repeatable events** - Set your events to repeat daily, weekly or monthly
- **Notifications** - Email notifications that being end 10 minutes before event
- **User Profiles** - Customizable avatars
- **Region Calendars** - Get calendar of public holidays depending on region specified during registration
## Technical Stack
### Backend
- **Node.js v18+** - JavaScript runtime
- **Express.js** - Web application framework
- **MongoDB Atlas** – Cloud database
- **Mongoose** - library to connect database and Node
- **Docker** – Containerization
- **Docker Compose** – Multi-container orchestration
- **bcrypt** - Password hashing
- **express-session** - Session management
- **nodemailer** - Email notifications
- **multer** - file uploading
- **Google Calendar Api** - public holidays
